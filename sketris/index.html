Sketris Change Log

Ver 0.2.2 - 3/22/24

  • Paster upgrades
    • On image paste, will try to guess the queue/hold if color detected
    • Can edit queue/hold directly in paster
  • General
    • Current tile under cursor is now highlighted
    • Change log displays on version change

Ver 0.2.1 - 3/6/24

  • Bug fix
    • Fixed desync issue when editing queue/hold in shared session (thanks Bennxt)

Ver 0.2.0 - 2/14/24

  • Happy Valentines Day!
  • Experimental collaborative mode
    • Host a session and send a link to friends!
    • Supports multiple people
  • General Improvements
    • Improved rendering for snappier gameplay
    • Added help popup for sketching controls
  • Valentines Easter Egg Puzzle

Ver 0.1.1 - 12/31/23

  • small new year's eve update
  • Bug fixes
    • instant soft drop preserved
    • queue editor uses monospace font for proper line wrapping
    • autocoloring glitches fixed
  • palette open by default for discoverability

Ver 0.1.0 - 12/18/23

  • The Big Christmas Update!
  • Cosmetic updates
    • Viewable Changelog
    • Side panel has tabs
    • Bag indicator reworked
    • Settings look nicer
    • Palette looks nicer
    • Everything looks nicer
  • Sketching Updates
    • Introducing the "Magic Marker" which autocolors pieces (even when disconnected!)
    • Middle mouse now activates garbage drawing mode! (idea cred: IndigoLaser)
      • draw up to create garbage
      • draw down to compress lines
    • Piece selections don't overwrite the board during drag now until mouse release
    • Removed color dropper
  • Data Import/Export
    • Sketris board states can be shared via link using the "data" query parameter in url
      • data can be copied as link or text
      • Current queue and hold can be edited
      • Queue regen can be enabled/disabled
    • Pasted screenshots can be interpreted as boards
      • images can be cropped to board size
      • automatic color guessing (tested with jstris/tetrio default colors)

Ver 0.0.10 - 11/12/23

  • ghost tiles can now be drawn by holding Alt

Ver 0.0.9 - 11/7/23

  • fixed compatibility issues with firefox
  • tile selection works with Control now

Ver 0.0.8 - 11/5/23

  • Eyedropper: sketching color can be changed by middle-clicking on a block

Controls

Left Click - Normal drawing (toggle on/off)

Right Click - Area drawing

Middle Click - Garbage drawing (drag up to create, down to remove)

Alt Click - Ghost drawing

Shift/Ctrl Click - Tile Selection (click and drag, Esc to deselect)

Paste images directly into window (Ctrl-V)

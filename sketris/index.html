Sketris Change Log

Ver 0.1.1 - 12/31/23

  • small new year's eve update
  • Bug fixes
    • instant soft drop preserved
    • queue editor uses monospace font for proper line wrapping
    • autocoloring glitches fixed
  • palette open by default for discoverability

Ver 0.1.0 - 12/18/23

  • The Big Christmas Update!
  • Cosmetic updates
    • Viewable Changelog
    • Side panel has tabs
    • Bag indicator reworked
    • Settings look nicer
    • Palette looks nicer
    • Everything looks nicer
  • Sketching Updates
    • Introducing the "Magic Marker" which autocolors pieces (even when disconnected!)
    • Middle mouse now activates garbage drawing mode! (idea cred: IndigoLaser)
      • draw up to create garbage
      • draw down to compress lines
    • Piece selections don't overwrite the board during drag now until mouse release
    • Removed color dropper
  • Data Import/Export
    • Sketris board states can be shared via link using the "data" query parameter in url
      • data can be copied as link or text
      • Current queue and hold can be edited
      • Queue regen can be enabled/disabled
    • Pasted screenshots can be interpreted as boards
      • images can be cropped to board size
      • automatic color guessing (tested with jstris/tetrio default colors)

Ver 0.0.10 - 11/12/23

  • ghost tiles can now be drawn by holding Alt

Ver 0.0.9 - 11/7/23

  • fixed compatibility issues with firefox
  • tile selection works with Control now

Ver 0.0.8 - 11/5/23

  • migrated project to use npm webpack

Ver 0.0.7 - 10/30/23

  • added bag postion indicator to queue
  • Eyedropper: sketching color can be changed by middle-clicking on a block

Ver 0.0.6 - 10/25/23

  • implemented color drawing

Ver 0.0.5 - 10/20/23

  • settings are now saved/loaded via browser cookie

Ver 0.0.4 - 10/20/23

  • Implemented highlighting blocks holding shift
  • highlighted blocks are draggable

Ver 0.0.3 - 7/12/23

  • Can update queue with custom piece order

Ver 0.0.2 - 7/10/23

  • Added board state history with undo/redo

Ver 0.0.1 - 7/8/23

  • Added basic board block sketching functionality

Ver 0.0.0 - 7/7/23

  • Playable implementation with DAS/ARR !
